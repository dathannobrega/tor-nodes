01 Capacidades

Detecção e análise de ameaças, do dado bruto à decisão.

Uma base de inteligência consolidada a partir de fontes oficiais e honeypots próprios.

Atualização automática

Cache de nós exit renovado a cada 12 horas e dados detalhados a cada 5 minutos, em background.

Respostas rápidas

Camada de cache em memória servindo os feeds em milissegundos, com rate limiting por IP.

Fontes confiáveis

Dados obtidos diretamente da API oficial do Tor Project (Onionoo e exit-addresses).

Honeypot Cowrie

URLs maliciosas capturadas em tempo real e filtradas por domínios legítimos.

Estatísticas detalhadas

Distribuição por país, flags, bandwidth e status de atividade dos relays.

API RESTful

Endpoints documentados em múltiplos formatos (JSON, TXT, RSS) prontos para automação.

const BASE = "https://cti.segark.com";

async function getRunningNodes() {
  const res = await fetch(`${BASE}/api/nodes/running`);
  if (!res.ok) throw new Error(`HTTP ${res.status}`);
  const { nodes } = await res.json();
  return nodes;
}

async function getMaliciousUrls() {
  const text = await (await fetch(`${BASE}/honeypot-urls.txt`)).text();
  return text.split("\n").filter((u) => u && !u.startsWith("#"));
}

getRunningNodes().then((n) => console.log(`Nós ativos: ${n.length}`));
<?php

$urls = file_get_contents("$base/honeypot-urls.txt");
$lines = array_filter(
    explode("\n", $urls),
    fn($l) => $l !== "" && !str_starts_with($l, "#")
);
echo "URLs detectadas: " . count($lines) . "\n";
?>
